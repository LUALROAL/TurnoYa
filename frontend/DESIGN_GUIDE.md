# TurnoYa Frontend - Design Guide (Tech Pro)

## Direction
Tech Pro: premium, modern, precise. Dark surfaces, high contrast accents, strong typography, compact UI.

## Typography (2 pro options)
Option A:
- Display: Space Grotesk (600-700)
- Body/UI: IBM Plex Sans (400-600)

Option B:
- Display: Sora (600-700)
- Body/UI: Work Sans (400-600)

Usage rules:
- Use Display font for headings and key numbers.
- Use Body font for paragraphs, labels, buttons, inputs.
- Avoid mixing both options at once.

## Color System
- Base: deep graphite + near-black surfaces
- Accent: electric cyan + ultraviolet for CTAs and active states
- Support: warm amber for warnings, cool green for success

## Layout & Spacing
- Grid: 4px base spacing (4, 8, 12, 16, 24, 32, 40, 48)
- Container max width (desktop): 1200px
- Mobile first: cards full width, stacked sections

## Shapes
- Radius: 12 (cards), 10 (inputs), 999 (chips)
- Borders: 1px with low-contrast outline

## Shadows
- Soft glow for focus and active elements (accent color)
- Subtle elevation for cards (dark shadow, low opacity)

## Components (baseline)
- Button primary: accent background, dark text, bold label
- Button secondary: transparent with outline
- Card: dark surface, 12 radius, top highlight line
- Input: dark surface, 10 radius, clear focus ring
- Badge/chip: compact, 999 radius, muted background

## Motion
- Page load: fade up 160-220ms
- Staggered lists: 60ms delay per item
- Hover: lift 2px + glow 100-140ms

## Imagery
- Use high-contrast photos or duotone overlays
- Avoid stocky, low-light images
